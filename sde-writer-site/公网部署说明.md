# 公网部署说明

如果你想在外地用手机访问，或者把这个智能体给其他人用，就不能只在电脑本地运行，需要部署到公网服务器。

## 推荐方式

可以部署到任意支持 Node.js 的平台，例如：

- Replit
- Render
- Railway
- Zeabur
- 自己的云服务器

这个项目没有第三方依赖，启动命令很简单：

```bash
node server.js
```

如果你使用 Render，请优先看：

```text
Render部署步骤.md
```

## 必须设置的环境变量

在部署平台后台设置：

```text
DEEPSEEK_API_KEY=你的 DeepSeek API Key
ADMIN_PASSWORD=你的管理员密码
PORT=5173
```

设置好 `DEEPSEEK_API_KEY` 后，网站后台会自动调用 DeepSeek。网页不会显示 API Key 输入框，用户也看不到你的 Key。

设置好 `ADMIN_PASSWORD` 后，网页会出现“邀请码”输入框和管理员发码区域。管理员可以生成 3 天、1 个月、3 个月三种邀请码。只有输入有效邀请码的人才能生成标题和文章。

## 重要提醒

如果你把网站公开给别人用，DeepSeek 调用费用会走你的 API Key。

建议后续再加：

- 每天生成次数限制
- 只允许指定用户使用

## Replit 部署流程简版

1. 新建一个 Node.js 项目。
2. 上传本文件夹里的全部文件。
3. 在 Replit 的 Secrets 或 Environment Variables 里设置：

```text
DEEPSEEK_API_KEY=你的 DeepSeek API Key
ADMIN_PASSWORD=你的管理员密码
PORT=5173
```

4. 启动命令填：

```bash
node server.js
```

5. 部署后，把公开网址发给用户。用户打开网址，输入邀请码即可使用。

## 本地和公网的区别

本地版：

```text
http://localhost:5173
```

只能在你的电脑上用，或者同一个 Wi-Fi 下用。

公网版：

```text
https://你的网址
```

手机在任何地方都能打开，也可以分享给别人。
